CircleCI Enterprise または CircleCI Server を更新するには、以下の手順を実行します。 このプロセスは、15 ～ 20 分程度で終わるシンプルなものです。

 

1. 管理コンソールのダッシュボードで、[View Update (更新の表示)] をクリックします。

 


2. [Install Update (更新のインストール)] をクリックします。 CircleCI Server の更新が開始されます。

 

 

注:
1. 更新の前に、https://circleci.com/ja/server/changelog/ で Server の更新履歴を確認してください。
2. 更新前に、現在の環境をバックアップしてください。 手順については、https://circleci.com/docs/ja/2.0/backup/#backing-up-circleci-data をご覧ください。
3. 実行中のジョブがないことを確認してください。

 

不明な点がありましたら、サポートにチケットを送信するか、support@circleci.com までメールにてお問い合わせください。
