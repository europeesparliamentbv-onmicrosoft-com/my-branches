The full and up-to-date table for the cost of each executor and size can be found here: https://circleci.com/pricing/#compute-options-table (under the "Powerful compute options" header).

Re-running with SSH will charge credits for the executor and resource class used. So if you re-run a job with SSH that originally ran a Docker Medium container (CircleCI's default size), you will be charged 10 credits/minute.
