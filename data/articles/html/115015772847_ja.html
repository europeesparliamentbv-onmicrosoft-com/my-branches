TZ 環境キーを使っても 2.0 macOS ビルドでは機能しません。 タイムゾーンを設定するには、構成に次のコマンドを追加する必要があります。

      # Set Timezone      - run:          name: set timezone          command: |            sudo systemsetup -settimezone "$TIMEZONE"

$TIMEZONE を使用するタイムゾーンに置き換えます。 サポートされているすべてのタイムゾーンのリストは、sudo systemsetup -listtimezones で確認できます。
