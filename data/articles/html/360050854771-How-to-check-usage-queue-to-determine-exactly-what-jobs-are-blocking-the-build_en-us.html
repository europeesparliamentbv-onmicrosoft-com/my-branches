If jobs in your pipeline are queued in usage queue and want to know which jobs are blocking your build, you can use the following API endpoint.

GET https://circleci.com/api/v1.1/project/{vcs}/{orgname}/{project}/{CIRCLE_BUILD_NUM}/usage-queue
  • vcs is the name of the VCS, either github or bitbucket
  • orgname is the name of the organization or username
  • project is the name of your repository
  • CIRCLE_BUILD_NUM the built-in environment variable associated with the specific build in question, which can be found in the UI

You can find the build URL with "build_url" key from the response.

For those on the Performance Plan getting stuck in the Usage Queue, you can request a concurrency increase by writing to billing@circleci.com. 

Please note that this is an undocumented API endpoint and might removed or modified without prior notice.
