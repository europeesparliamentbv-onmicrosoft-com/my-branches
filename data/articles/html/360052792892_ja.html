組織の CircleCI Server プランを確認するには、以下の API エンドポイントを使用します。 トライアルの終了日や組織名など、さまざまなパラメーターが表示されます。

 

ライセンスの詳細情報を確認する場合は、こちらで紹介しているエンドポイントを確認するか、http://<fqdn>/admin/license にアクセスしてください。

 

<circleci_url>/api/v1/admin/organization/:username/plan

 

 

注: このエンドポイントを実行するには、管理者である必要があります。

 

不明な点がありましたら、サポートにチケットを送信するか、support@circleci.com までメールにてお問い合わせください。
